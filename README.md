# Pemetaan-2D-Kota-Virtual-Dengan-Simulasi-Navigasi-Kendaraan
Kota nyata memiliki jaringan jalan kompleks, saling terhubung, dan tidak selalu lurus. Permasalahan: bagaimana membangun peta kota virtual 2D yang logis serta memastikan tidak ada jalan terisolasi. Solusi: membangun peta secara prosedural dengan jalan melengkung, saling terhubung, dan dapat disimulasikan interaktif.
